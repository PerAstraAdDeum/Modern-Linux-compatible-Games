# Games compatible with modern Linux

A curated list of games that run on Linux and neither require x86 compatibility nor XWayland.
Note: All of the following games were tested with `ia32_emulation=0` Kernel argument. They were installed using Heroic Games Launcher's download system, which unlike GOG's offline installers does not require x86 emulation.
